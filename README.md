## Analysis and Dashboard Creation in Power BI 
In this project, I have cleaned and analyzed data from different CSV files and then converted all the insights into a Power BI dashboard. All the processes in this project has been done in Power BI and it has helped
me to gain experience on how to work on huge datasets only using Power BI.
## Process
- Aggregated and Prepped 7 CSV files which contained 3 years of sales data in Power BI, cleaned the data to reduce errors by 25%
- Utilized techniques to create 2 interactive dashboards, which included around 11 key performance indicators(KPIs) to analyze sales and product performance.
- Leveraged features such as drill-down capabilities, trend analysis, and matrix to provide actionable insights.
